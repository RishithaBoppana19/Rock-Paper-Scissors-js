# ✊🖐✌ Rock Paper Scissors Game

A fun and interactive Rock-Paper-Scissors game made with HTML, CSS, and JavaScript. Challenge the computer and see who wins!


## 🎮 Features

- Simple UI with clickable choices: Rock, Paper, or Scissors
- Score tracking for both Player and Computer
- Real-time result display (win/lose/draw)
- Responsive and stylish layout
- Randomized computer moves


## 📂 Project Structure

─ index.html # HTML structure of the game
─ style.css # Styling for layout and visuals
─ app.js # Game logic and interactivity
─ images/
─ rock.png
─ paper.png
─ scissors.png

> 📌 Make sure the `images` folder exists and includes the images for rock, paper, and scissors in the correct filenames.


## 🚀 How to Play

1. Open `index.html`** in any web browser.
2. Click on one of the choices: Rock, Paper, or Scissors.
3. The computer randomly picks its choice.
4. The result is displayed along with updated scores.
5. Keep playing and have fun!
